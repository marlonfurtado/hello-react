# Hello React !
> A application to test and learn about React

## What is:
Is a simple chat application using React.js and Firebase as a backend data service. 

## Running

``` bash
# install dependencies
npm install

# serve at localhost:3000
npm start

# run tests
npm test

# build for production
npm run build

```
